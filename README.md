# Super Mario Clone JS
Super Mario Clone using Vanilla JavaScript.  

Live version deployed to heroku at: https://super-mario-clone-js.herokuapp.com/

## How to run
Installing the dependencies
```
npm install
```
Running the program
```
npm run start
```
This by default uses port 5000.
